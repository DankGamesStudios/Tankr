# Tankr
Arcade Tank Game

[Play the game](https://dankgamesstudios.github.io/Tankr/)

### Contributing

```bash
npm install
npm run-script server:dev
```
Then start playing or start writing code.


### Updating demo app
```bash
npm run-script build:dist
npm run-script deploy
```
